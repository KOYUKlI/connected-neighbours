## Démonstration prévue

1. Connexion locale de développement via le client web.
2. Création et consultation d'une annonce de service.
3. Acceptation d'un service payant avec réservation des points et création de contrat.
4. Signature puis clôture du contrat avec transfert des points.
5. Consultation de Swagger sur `/docs`.
6. Exécution des tests unitaires API.

La partie JavaFX est volontairement gardée pour un lot séparé.
